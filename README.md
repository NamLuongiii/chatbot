# Solashi Chatbot Widget

## Introduction

---

## Features

- Popup chatbot displayed on the landing page
- Easy integration with a single script file
- Framework-agnostic (works with plain HTML)
- Build once, reuse across multiple pages
- Lightweight and performance-optimized
- SEO-friendly

---

## Requirements

- Node.js >= 16
- npm or yarn
- A website that allows inserting `<script>` tags

---

## Integration Guide

- set env variables in the.env file
- npm i
- npm run build
- copy dist folder to your website
- add a script tag to your website <script src="dist/index.js"></script>
- add a CSS link to your website <link rel="stylesheet" href="dist/index.css">
